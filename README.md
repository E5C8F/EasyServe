# 实时在线聊天程序

> ⚠ **重要通知：该项目目前已暂停更新和维护**

## 项目简介

一个免费的实时在线聊天程序，无需复杂配置，下载即可使用。适合个人或小团体快速搭建聊天环境。

## 主要功能

- **实时在线聊天**：支持多用户实时文字交流
- 简洁直观的用户界面
- 自动连接和消息推送

## 系统要求

- Windows 10
- Windows 11

## 安装说明

**无需安装，直接运行**

1. 下载程序压缩包
2. 解压到任意文件夹
3. 双击运行主程序即可使用
